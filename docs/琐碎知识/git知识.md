## 以前已经学过一些基础的了 这里是放实际遇到的问题
1. 推送到远程： 
    ~~~text
   先创建远程仓库 复制ssh  
   然后 推送的时候定义远程仓库
    ~~~
2. 定义远程仓库时 的名称origin
    ~~~text
   因为可以本地 1 对 多个远程 所以得给他们起名字以区分
   默认origin
   ~~~
3. 不要远程仓库了：
    ~~~text
   先控制台 取消关联
   git remote remove origin
   
   然后删除github 库就行
    ~~~
4. 本地修正提交 再推；
    ~~~text
   迁出 变基
   ~~~
