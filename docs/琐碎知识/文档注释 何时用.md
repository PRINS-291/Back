### 普通注释
 普通注释是给程序员看的，用于解释代码逻辑，不会被编译器和工具特殊处理。

### 文档注释（Javadoc）
文档注释是特殊的注释，以 /** 开头，会被 javadoc 工具解析生成API文档。
    
### 常用标签：

|     标签      |  用途   |           示例           |
|:-----------:|:-----:|:----------------------:|
|   @param    | 参数说明  |    @param name 用户名     |
|   @return   | 返回值说明 |      @return 操作结果      |
|   @throws   | 异常说明  |  @throws IOException   |
|    @see     | 参考链接  |    @see OtherClass     |
|   @since    | 引入版本  |       @since 1.8       |
| @deprecated |  已过时  |   @deprecated 使用新方法    |
|   {@code}   | 代码片段  |   {@code int x = 5;}   |
|   {@link}   | 内联链接  | {@link #otherMethod()} |

### 最佳实践总结
| 方面 |        文档注释        |    普通注释     |
|:--:|:------------------:|:-----------:|
| 受众 |    API使用者、其他开发者    | 代码维护者、团队成员  |
| 位置 |    类、接口、方法、字段前     |  代码行内、方法内部  |
| 工具 | 被javadoc解析生成HTML文档 |  仅出现在源代码中   |
| 内容 |      是什么、怎么用       | 为什么这么做、如何实现 |
| 维护 |     需要与代码同步更新      |  可以临时性、解释性  |
