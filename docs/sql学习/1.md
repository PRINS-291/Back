## sql： structure query language
~~~text
1. DDL：数据定义语言 用来操作数据库 create alter drop
2. DML：数据操作语言 用来操作数据库中的数据 insert update delete
3. DCL：数据控制语言 用来操作访问权限 安全级别 grant deny
4. DQL：数据查询语言 用来查询数据 select
~~~

## 三大范式
~~~text
1. 每一列不可再分
2. 1的基础上 完全依赖主键 （多个复合而成 绝不能只依赖一部分
3. 2的基础上 非主键列必须与主键列直接相关
~~~

~~~text
1. 整数类型：tinyint smallint mediumint int bigint 12348
2. 浮点数类型：float double 48 
3. 定点数类型：decimal(M,D) M:数据长度 D：小数点后的长度 M+2 8
4. 字符串类型：char(n) varchar(n) 固定为n 实际长度+1
5. 文本类型：tinytext text mediumtext longtext 
6. 日期与时间： year date time datetime timestamp 1 4 3 8 4 
7. 二进制类型: tinyblob blob mediumblob longblob
~~~

### 基本操作
~~~text
一. 数据库
1. 创建数据库
    create database 数据库名称 character set utf8 collate utf8_bin(utf8_general_ci 不区分大小写);
    
    显示数据库创建时的语句
    show create database 数据库名称;
    
    规避关键字用反引号 `INT`
    
2. 删除数据库
    drop database db1;
3. 展示所有数据库
    show databases;
4.修改数据库字符集
    alter database 数据库名称 character set 字符集;
5.使用 切换数据库
    use 数据库名称;
6. 查看当前使用的数据库
    select database();

二.数据表
1.创建
     create table student(
     id int,
     name varchar(20),
     gender varchar(10),
     birthday date
     );
2.看所有表
    show tables;
3.看表的结构信息
    desc 表;
4.修改表名
    alter table 表 rename to 新名字;
~~~
